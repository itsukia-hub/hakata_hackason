ハルシネーション（幻覚）

AIが「もっともらしい嘘」をつく現象

  • 存在しない本や論文を紹介してしまう
  • 実在しないURLを生成してしまう
  • 事実と異なる情報を自信満々に回答する
  • 一見正しく見えるため気づきにくいのが厄介

重要な情報は必ず裏取りしましょう

ハルシネーション対策

AIの出力を信じすぎない ― 3つの防御策

  1. 裏取りの習慣
    重要な数値・固有名詞・URLは必ず元ソースで確認
  2. RAG（検索拡張生成）の活用
    NotebookLMのように、手持ちの資料だけを根拠にさせるとハルシネーションが激減
  3. 複数のAIでクロスチェック
    ChatGPT / Claude / Gemini で同じ質問をし、回答が食い違ったら要注意

AIは「頼れるアシスタント」。ただし最終判断は自分でやる

コンテキストの重要性

背景情報を与えると回答品質が変わる

悪い例

「いい感じのメールを書いて」

良い例

「取引先のA社に、来週の会議のリスケをお願いするメールを書いて。丁寧なトーンで」

RAG（検索拡張生成）: AIに外部データを参照させて精度を上げる仕組み
NotebookLMはこの仕組みを活用しています

チャット型からエージェント型へ

エージェント型はチャット機能を含んだ「上位の存在」です

チャット型AI

  • 1回の質問に1回の回答
  • ユーザーが対話を主導
  • テキストベースのやり取り

例: ChatGPT、Claudeのチャット

エージェント型AI

  • チャット機能 ＋ 自律的な行動
  • ファイルの読み書きも自動
  • コード生成 → テスト → 修正まで一気通貫

例: Claude Code、Cursor、
Gemini Code Assist

エージェント型 = 「チャットもできるし、自分で考えて動くこともできる」AI

✨ Gemini Code Assist

VSCodeで使える無料のAIエージェント

  • Googleの個人用アカウントで無料利用可能
    ※会社アカウント（Workspace管理下）では利用できない場合があります
  • VSCodeの拡張機能として動作
  • 3つの機能:
    • コード補完 ― 入力中に候補を提示
    • チャット ― コードについて質問
    • エージェントモード ― 自律的にコードを生成・編集
      ※本ワークショップでは「エージェントスイッチ」はONにせず使います（トークン節約のため）

Gemini Code Assist ― 使うときのTips

  • 💤 回答が遅いとき
    一度サインアウトして再度サインインすると改善する場合があります
  • 🔒 オプトアウト設定
    利用設定で「コードをGoogleの学習に使わせない（オプトアウト）」が可能。機密情報を扱う場合は必ずオフに
  • ❌ 生成結果のリジェクト（却下）
    AIが提案した変更が気に入らないときは「拒否（Reject）」して再依頼できます。無理に採用せず、何度でも作り直せます

困った時のプロンプトはこちら 
※ 既存の index.html / style.css / game.js を残したまま、以下の機能を追加してください。

プレイヤーをマウスで動かせるようにしてください。
- マウスカーソルの位置を記録する（mousemoveイベント）
- 毎フレーム、プレイヤーがマウスの方向に向かって移動する
- 移動スピードは 3.5
- フィールドの外には出ないように制限する
- requestAnimationFrameを使って毎フレーム更新する

動き方のイメージ：
マウスを動かすとプレイヤーが
ゆっくり追いかけてくる感じです。
想定コードはこちら（game.js 新規記述） 
// HTML要素の取得
var field = document.getElementById('field');
var playerEl = document.getElementById('player');

// ゲーム設定
var W = 800, H = 600;
var playerR = 14;
var playerSpeed = 3.5;

// プレイヤー位置・マウス位置（中央からスタート）
var playerX = W / 2;
var playerY = H / 2;
var mouseX = W / 2;
var mouseY = H / 2;

// マウス位置を記録
field.addEventListener('mousemove', function (e) {
  var rect = field.getBoundingClientRect();
  mouseX = e.clientX - rect.left;
  mouseY = e.clientY - rect.top;
});

// 毎フレームの更新処理
function update() {
  var dx = mouseX - playerX;
  var dy = mouseY - playerY;
  var dist = Math.sqrt(dx * dx + dy * dy);
  if (dist > 2) {
    playerX += (dx / dist) * playerSpeed;
    playerY += (dy / dist) * playerSpeed;
  }
  // フィールド内に制限
  playerX = Math.max(playerR, Math.min(W - playerR, playerX));
  playerY = Math.max(playerR, Math.min(H - playerR, playerY));
  // 位置を反映
  playerEl.style.left = (playerX - playerR) + 'px';
  playerEl.style.top = (playerY - playerR) + 'px';
}

// メインループ
function gameLoop() {
  update();
  requestAnimationFrame(gameLoop);
}
gameLoop();

// 冒頭（要素取得）に追加
var gameoverScreen = document.getElementById('gameover-screen');

// ゲーム設定に追加
var state = 'playing'; // 'playing' / 'gameover'

// 当たり判定の関数（update の前に追加）
function isHit(x1, y1, r1, x2, y2, r2) {
  var dx = x1 - x2;
  var dy = y1 - y2;
  return Math.sqrt(dx * dx + dy * dy) < r1 + r2;
}

// update() の先頭に追加（動作停止）
if (state !== 'playing') return;

// update() の末尾（鬼の位置反映の後）に追加
if (isHit(playerX, playerY, playerR, oniX, oniY, oniR)) {
  state = 'gameover';

function startGame() {

  oniX = 100; oniY = 100;

// 冒頭（要素取得）に追加
var timeDisplay = document.getElementById('time-display');
var clearScreen = document.getElementById('clear-screen');

// ゲーム設定に追加
var timeLeft = 30;
var timer = null;

// HUD更新関数（game.js の末尾などに追加）
function updateHUD() {
  timeDisplay.textContent = '残り ' + timeLeft + '秒';
}

// startGame 関数をタイマー起動版に差し替え
function startGame() {
  state = 'playing';
  playerX = W / 2; playerY = H / 2;
  oniX = 100; oniY = 100;
  timeLeft = 30;
  updateHUD();
  gameoverScreen.style.display = 'none';
  clearScreen.style.display = 'none';

  clearInterval(timer);
  timer = setInterval(function () {
    if (state !== 'playing') return;
    timeLeft--;
    updateHUD();
    if (timeLeft <= 0) {
      state = 'clear';
      clearInterval(timer);
      clearScreen.style.display = 'flex';
    }
  }, 1000);
}

var scoreDisplay = document.getElementById('score-display');

// ゲーム設定に追加
var items = [];
var score = 0;

// アイテムを1個追加する関数
function addItem() {
  var div = document.createElement('div');
  div.className = 'item';
  var x = 30 + Math.random() * (W - 60);
  var y = 40 + Math.random() * (H - 70);
  div.style.left = (x - 10) + 'px';
  div.style.top = (y - 10) + 'px';
  field.appendChild(div);
  items.push({ x: x, y: y, r: 10, el: div });
}

// updateHUD を以下に差し替え
function updateHUD() {
  timeDisplay.textContent = '残り ' + timeLeft + '秒';
  scoreDisplay.textContent = 'スコア: ' + score;

// update() の末尾に追加（アイテム取得判定）
for (var i = items.length - 1; i >= 0; i--) {
  if (isHit(playerX, playerY, playerR, items[i].x, items[i].y, items[i].r)) {
    score += 100;
    updateHUD();
    items[i].el.remove();
    items.splice(i, 1);
  }

// startGame の冒頭に「アイテム初期化」を追加
